CH340G

安いのにDTR,RTSピン付き&SOPなのでハンダ付けしやすいシリアル変換ICのCH340Gを使う方法
  • 入手方法
    ちっちゃいものくらぶさんから完成品(300円+送料)とIC単体(100円+送料)で入手可能。データシートとドライバもあり
    eBayでIC単体で150円くらい、完成品のモジュールが180円くらい。送料無料のものが多い
    AliExpressだとIC単体で60円くらいで入手可能。安いものは送料が高いので注意すること!
    aitendoで500円

  • 必要な部品
    12MHzの水晶振動子 x 1個
    22pFのコンデンサ x 2個
    お好みのタイプのUSBコネクタ x 1個
    SOP→DIP変換基板 300mil用 秋月のＳＯＰ１６ピン（１．２７ｍｍ）ＤＩＰ変換基板　（５枚入）など

  • 配線方法 (mbed用にCTSをRTSに接続してあります)
    なかなか認識しない人へ→CH340GのVCCとGND間にパスコン(104)を挿入するだけで安定します



    秋月MiniUSB変換基板のピンアサイン

  • シリアルBitBangでAVRライター化 (まだ試験段階です 2014/7/27)
    FTDI FT232RLや230X,231XのBitBangなら良く知られているし、フロントエンドツールも充実している。
    汎用シリアルでもBitBangが使えればAVRライターになるし、FTDIのライブラリをリンクしたavrdudeを探す必要もなくなる(←これ重要)
    avrdude.conf にはtype=serbbという指定ができる
    serbbがシリアルBitBangを意味することを suzakulab さんから教えてもらった

    接続方法は avrdude.conf に書かれているとおり、RTS→RESET,DTR→SCK(Arduno D13),TXD→MOSI(Arduino D11),CTS→MISO(Arduino D12)に接続する
    下記はdasaにパラメータを上書きしたもの

    avrdude.conf

    programmer
      id    = "dasa";
      desc  = "#serial port banging, reset=rts sck=dtr mosi=txd miso=cts";
      type  = serbb;
      connection_type = serial;
      reset = ~7; # RTS
      sck   = ~4; # DTR
      mosi  = ~3;   # TXD
      miso  = ~8;   # CTS
    ;
    avrdudeで -P dasa とプログラマを指定すればOK,CH340GではISP CLOCK Delay(-iオプション)を150msにすると安定した
    下記はATmega328Pを自動認識してヒューズを読み込んだところ
    なぜかavrdude-GUIにはdasaが表示されない (未解決)
    avrdude -c dasa -p auto -P COM3 -B 100000 -i150 -u -e -U flash:w:ATmegaBOOT_168_atmega328_pro_8MHz.hex:a でブートローダー書込
    150msのディレイのためかブートローダーの書込みに45分もかかっている
    書込風景



  • 動作中の様子


    コンデンサは33pFでも動作した
     

    RXD,TXDライン直結でLEDが点灯する。データ送受信も問題ないようだ。Rは4.7k


    珍しい筒状のクリスタルを使ってみた


    22pFのセラコンは秋月のP-3620


    クリスタルは普通のHC-49/Sタイプ。秋月だとP-00543
ミスった時のNG集
  • よくあるUSBが認識されないパターン。D+/D-やセラコンの接続が間違っているとこうなります
  • 最悪のエラーはこんな画面でWindowsがシャットダウンします
